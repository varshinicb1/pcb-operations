## Project Identity
This is **Prakash Coach Builders Operations Management System** ‚Äî a Flutter + Firebase app for bus manufacturing workforce management. Target: Android APK + Web, 5+ branches, 1000+ employees.

## Critical Rules

### Never Touch These
- `lib/firebase_options.dart` ‚Äî auto-generated by flutterfire, contains real API keys
- `android/app/google-services.json` ‚Äî auto-generated
- `android/app/src/main/res/mipmap-*/ic_launcher.png` ‚Äî auto-generated by flutter_launcher_icons
- `*.g.dart` files ‚Äî auto-generated by build_runner, run `dart run build_runner build` to regenerate

### Always Run After Changes
```bash
flutter analyze          # Must show 0 errors before commit
dart run build_runner build  # Regenerate .g.dart files if models changed
```

### Architecture Constraints
- **State**: Riverpod only, no BLoC/Provider/GetX
- **Navigation**: GoRouter only, no Navigator.push for main routes
- **Models**: `json_serializable` for JSON, hand-written `copyWith`
- **Firebase**: Spark free tier ‚Äî no Storage, no Functions, no Phone Auth
- **Selfies**: base64 in Firestore `attendance.selfieUrl`, NOT in Storage
- **Role checks**: `get()` on employee document in Firestore rules, NOT custom claims
- **Theming**: Colors in `app_colors.dart`, theme in `app_theme.dart`, no inline colors
- **Reusable widgets**: Add to `lib/core/widgets/`, use from there

### File Naming & Location
- Models: `lib/data/models/<name>.dart`
- Repos: `lib/features/<feature>/data/<name>_repository.dart`
- Screens: `lib/features/<feature>/presentation/<name>_screen.dart`
- Core widgets: `lib/core/widgets/<name>.dart`
- Utils: `lib/core/utils/<name>.dart`

### Provider Pattern
```dart
// Repository class with Firebase dependencies
class SomeRepository {
  final FirebaseFirestore _firestore;
  SomeRepository(this._firestore);
  // methods here
}

// Provider (in same file as repository)
final someRepositoryProvider = Provider<SomeRepository>((ref) {
  return SomeRepository(FirebaseFirestore.instance);
});

// Data providers
final someDataProvider = StreamProvider.autoDispose<SomeType>((ref) {
  return ref.watch(someRepositoryProvider).watchData();
});
```

### UI Pattern
```dart
class SomeScreen extends ConsumerWidget {  // or ConsumerStatefulWidget
  const SomeScreen({super.key});
  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final data = ref.watch(someDataProvider);
    return data.when(
      loading: () => const CircularProgressIndicator(),
      error: (e, _) => ErrorWidget(e.toString()),
      data: (d) => ActualContent(d),
    );
  }
}
```
